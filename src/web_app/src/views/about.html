Bimba is a simple application that will help you move between any two points in the public transportation system of the Poznań agglomeration.

Simply tell it where you want to start and finish your journey, and it will pick the best route for you.

Frequently Asked Questions:

Where does Bimba take its data from?

Bimba uses the data published by the City Transport Bureau in Poznań (Zarząd Transportu Miejskiego w Poznaniu). The data is then processed by the application and stored in a graph database for the ease of access.

Can I contribute to the application?

Of course you can! Bimba is an open source application that can be used and modified by anyone. For more information, see the license.

How can I report a bug or suggest a new feature?

You can do this by creating a new issue on the application's GitHub page.
